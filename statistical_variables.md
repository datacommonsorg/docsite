# Statistical Variables

Many of the Data Commons APIs deal with Data Commons nodes of the type [StatisticalVariable](https://browser.datacommons.org/kg?dcid=StatisticalVariable). The following variables have human-readable dcids:

-   [TotalPopulation](https://browser.datacommons.org/kg?dcid=TotalPopulation)
-   [MalePopulation](https://browser.datacommons.org/kg?dcid=MalePopulation)
-   [FemalePopulation](https://browser.datacommons.org/kg?dcid=FemalePopulation)
-   [MarriedPopulation](https://browser.datacommons.org/kg?dcid=MarriedPopulation)
-   [DivorcedPopulation](https://browser.datacommons.org/kg?dcid=DivorcedPopulation)
-   [NeverMarriedPopulation](https://browser.datacommons.org/kg?dcid=NeverMarriedPopulation)
-   [SeparatedPopulation](https://browser.datacommons.org/kg?dcid=SeparatedPopulation)
-   [WidowedPopulation](https://browser.datacommons.org/kg?dcid=WidowedPopulation)
-   [MedianAge](https://browser.datacommons.org/kg?dcid=MedianAge)
-   [MedianIncome](https://browser.datacommons.org/kg?dcid=MedianIncome)
-   [BelowPovertyLine](https://browser.datacommons.org/kg?dcid=BelowPovertyLine)
-   [HousingUnits](https://browser.datacommons.org/kg?dcid=HousingUnits)
-   [Households](https://browser.datacommons.org/kg?dcid=Households)
-   [TotalCrimes](https://browser.datacommons.org/kg?dcid=TotalCrimes)
-   [UnemploymentRate](https://browser.datacommons.org/kg?dcid=UnemploymentRate)
