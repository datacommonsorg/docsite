How To Sort

To sort columns of data from the API, we suggest to use the "filtered view" in Google Sheet. The steps are:

  • Populate an array formula using the {"HEADER"; formula} syntax.
  • Turn on filter views (Data > Filter views).
  • Click the reverse triangle on each column to sort by that column (and exclude row 1 header).
